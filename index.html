ABOUT

Hi, I'm Po Rith – and this is my GitHub portfolio.

I'm a designer turned developer, currently living in Oakland, California. I made this website as an opportunity to showcase some of my most recent learning and personal projects in modern web development. Coming from a background in design and the arts (see drawing), I've been able to take the last year to teach myself how to code. Amid some frustrations and many a nights, I can say that I have gained but a strong understanding of HTML/CSS, Javascript, and of an on-going variety of Javascript frameworks and APIs.

Because this website is a work in progress, please feel free to return and see new projects.

Thank you much for visiting :-)

FEATURED WORK

Oceans App

Oceans is a single-page application displaying information of world oceans by their sea-bodies. Data is fetched in real-time from a mocked-backend returning ocean information including tide, temperature, and other conditions. Features also include push notifications, offline-first, and accessibility.

  • HTML/CSS
  • Bootstrap
  • Javascript
  • Angular JS
  • Service Workers API
  • Dexie.js
  • Angular-route.js
  • Angular-mock.js

Bartor

Bartor is a realtime transportation application, functioning as a schedule tracker that utilizes offline functionality with Service Workers and IndexedDB. Requests are fetched live through BART API, returning exact fare and travel durations from station to station.

  • Bootstrap
  • Javascript
